# 📊 全球投资看板

投资小白友好型本地投资看板，展示全球股市行情、A股板块、财经新闻、大佬观点。

## 快速开始

```bash
# 1. 启动后端
cd ~/invest-dashboard
python3 start.py

# 2. 浏览器打开
# http://localhost:8888
```

## 功能模块

| 模块 | 数据源 | 说明 |
|------|--------|------|
| 全球指数 | 新浪财经 | A股5个 + 港股2个，30秒自动刷新 |
| A股板块 | 本地数据 | TOP10涨幅/跌幅排行（东方财富接口暂不可用） |
| 财经新闻 | 东方财富 | 20条最新新闻，来源+摘要+标签 |
| 大佬观点 | 静态数据 | 巴菲特、达利欧、黄仁勋、蔡崇信、马斯克 |
| 自选股 | 本地存储 | 支持A股/港股/美股，LocalStorage持久化 |
| 时钟 | 本地 | 多时区 + 各市场交易状态 |

## 技术细节

- **后端**: Python3 内置 http.server，端口 8888
- **前端**: 纯 HTML/CSS/JS，无框架
- **数据缓存**: 30秒 TTL，减少 API 调用
- **自动刷新**: 30秒（交易时段）

## 常见问题

**Q: 为什么某些指数显示周末休市？**
A: 今天是周末（6月7日），所有市场休市，显示的是上次收盘数据。

**Q: 板块数据是静态的？**
A: 是的，东方财富 API 因网络限制暂不可用。网络恢复后会自动切换。
